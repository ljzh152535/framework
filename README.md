## golang 框架工具

### 初始化推送到github
```
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:ljzh152535/framework.git
git push -u origin main
```

### 更新代码推送github
```
git add .
git commit -am "first commit"
git push -u origin main
```
